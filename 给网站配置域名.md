# 给网站配置域名

这篇博客只是为了方便日后本人日后配置，所以比较简陋。

1. 有一个公网IP。
2. 有一个域名（在`name.com`诸如此类的网站购买）。
3. 去`cloudflare`配置域名解析。
4. `cloudflare`会提供它的域名解析服务器网址，我们得去购买域名的网站比如`name.com`修改域名解析服务器的网址为`cloudflare`所提供的。
5. 在服务器配置`nginx`反向代理。
6. （可以在`cloudflare`配置`cdn`服务，这样就不能通过域名查到ip了
